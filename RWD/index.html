Introduction

Le Responsive Web Design est une approche de conception Web qui vise à offrir une expérience utilisateur optimale sur tous les appareils, des ordinateurs de bureau aux smartphones et tablettes.

Aujourd’hui, nous pouvons utiliser principalement trois méthodes pour répondre aux défis amenés par les différentes tailles d’écran. On peut :
- Créer une application dédiée pour les mobiles ;
- Créer une « copie mobile » de notre site en utilisant l’initiative AMP (« Accelerated Mobile Pages ») de Google ;
- Utiliser les Media Queries ou requêtes media.

Les tutoriels

Comment rendre votre site web responsive? Je vais vous donner les étapes avec le tutoriel sur youtube pour vous aider de mieux comprendre. Si cela semble trop difficile à réaliser, vous pouvez toujours soit engager un développeur WordPress, soit simplement vous assurer que votre thème est déjà responsive.

1. Définissez vos plages de media query (points de rupture responsives)
2. Taille des éléments de mise en page avec des pourcentages ou création d'une grille CSS.
3. Mettre en œuvre des images responsives.
4. Une typographie responsive du texte de votre site web.
5. Tester le responsive.
